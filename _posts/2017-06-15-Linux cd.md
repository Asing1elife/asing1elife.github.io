> cd-目录跳转  

## 系统根目录
```
cd /
```

## 返回上级目录
```
cd ..
```

## 当前用户主目录
```
cd  或  cd ~
```

## 跳转指定目录
```
cd ~/Document/innovaee/project
```

## 返回进入此目录之前所在的目录
```
cd -
```

## 用上一个命令作为参数使用
```
cd !$
```
