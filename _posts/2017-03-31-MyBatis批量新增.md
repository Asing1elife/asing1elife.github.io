> 通过循环 values 语句实现批量新增数据  

```xml
<insert id="saveTimetables">
 INSERT INTO pt_timetable (
  id,
  lessonNo
 )
 VALUES
  <foreach collection="list" item="item" separator=",">
   (
    #{item.id},
              #{item.lessonNo}
          )
  </foreach>
</insert>
```
