> NPM是Node.js的包管理模块  

## 操作
### 安装
```sh
sudo npm install -g n
```

### 安装最新稳定版 
```sh
sudo n stable
```

### 更新
```sh
sudo npm install npm@latest -g
```

### 清除缓存
```sh
npm cache verify
```
